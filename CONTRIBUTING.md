# Contributing

Thank you for your interest in contributing to this portfolio project! While this is primarily a personal portfolio, contributions and suggestions are welcome.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/portfolio.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit your changes: `git commit -m 'Add some feature'`
6. Push to the branch: `git push origin feature/your-feature-name`
7. Open a Pull Request

## Development Guidelines

- Follow the existing code style and patterns
- Use TypeScript for all new code
- Ensure all components are accessible (ARIA labels, semantic HTML)
- Test your changes locally before submitting
- Update documentation if needed

## Code Style

- Use functional components with hooks
- Follow React best practices
- Use Tailwind CSS for styling
- Keep components small and focused
- Add proper TypeScript types

## Commit Messages

Please write clear commit messages:
- Use present tense ("Add feature" not "Added feature")
- Be descriptive but concise
- Reference issues if applicable

## Questions?

Feel free to open an issue for any questions or suggestions!
